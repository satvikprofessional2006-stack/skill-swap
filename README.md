# SkillSwap 🌿 | Trade What You Know, Learn What You Don't

## 📖 The Product Story
SkillSwap was born from a simple observation: every student in our college is an expert at something but a beginner at something else. While one student excels at **Python**, another might be a **UI/UX** wizard. 

However, there was no centralized platform to connect these students for a mutual exchange of knowledge. **SkillSwap** solves this by providing a peer-to-peer learning marketplace where "Knowledge is the Currency." No money is exchanged—just skills.

## 🚀 Key Features
- **Intelligent Matching**: Finds peers who want to learn exactly what you can teach.
- **Full CRUD Profile**: Users can manage their skills, bio, and shared Google Meet links.
- **Real-time Communication**: A robust chat system with low-latency polling to discuss doubts.
- **Search & Discovery**: Filter through student profiles based on specific skill keywords.
- **Responsive Design**: Fully optimized for mobile and desktop for learning on the go.

## 🛠️ Tech Stack
- **Frontend**: HTML5 (Semantic), CSS3 (Modern Flexbox/Grid), Vanilla JavaScript.
- **Backend**: Node.js & Express.js.
- **Database**: JSON-based persistent storage (File System).
- **Architecture**: RESTful API design with asynchronous data fetching.

## 📈 Technical Highlights (For Viva)
- **API Latency Analysis**: Custom middleware implemented to track response times.
- **Security**: Data sanitization and "Safe Object" patterns to protect user passwords.
- **Performance**: Optimized chat rendering engine to prevent UI flickering.
- **Responsive UI**: Custom media queries without external heavy libraries like Bootstrap.

## ⚙️ Local Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the server with `node server.js`.
4. Open `http://localhost:3000` in your browser.
